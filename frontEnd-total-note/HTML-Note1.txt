=========  chapter3  HTML简介
HTML(hypertext markup language) 超文本标记语言
网页就是纯文本编写的，纯文本中只能保持文本内容，格式化、视频、音频都不能保存。
文件扩展名 改成 html 即可
超文本：文本 + 音频、视频等
标记：<p><h1></h1>等等标签
网页的标准格式：
<html 根标签>
  <head 子标签（设置网页的头部信息，不会在网页中显示，帮助浏览器解析网页）>
    <title （网页的标题栏）>标题</title>
  </head>
  <body 子标签（页面中所有的可见内容）>
    <h1></h1>
  </body>
</html>

=========  chapter8  乱码问题
出现的原因：在计算中保存的任何内容最终都保存为二进制的编码内容，包括网页中的内容。
在读取内容时，需要将二进制编码转化为正确的内容。
编码：依据规则，将字符转化为二进制编码的过程
解码：依据规则，将二进制编码转化为字符的过程
规则：字符集（编码和解码采用的规则）
产生乱码的原因：编码和解码采用的字符集不同
常见的字符集：ASCII（美国，7位，128）、ISO-8859-1（欧洲，8位，256）、GBK（中国）、UTF-8（万国码，支持所有的文字，unicode）、GB2312（中文系统的默认编码）
在中文系统的浏览器中，默认使用GB2312进行解码的，所以会出现乱码问题。
告诉浏览器网页采用的编码字符集：<meta（设置网页的元数据：字符集、关键词等等，自结束标签）></meta>
<meta charset="utf-8" /> 自结束标签
ANSI 自动编码，根据系统的默认编码方式保存文件。

=========  chapter9  常用的标签
内容的标题标签 <h1><h2>...<h6>，文字大小并不关心，通过css控制表现形式，关心的是标签的语义，使用的标签都说是语义化标签。
6级标题中，<h1>最重要，表示网页的主要内容，重要性依次降低
对于搜索引擎来说，h1的重要性仅次于title，搜索引擎检索完title，会立即查看h1中的内容
h1标签非常重要，会影响页面在搜索引擎中的排名，页面只能写一个h1
一般页面中标签只使用h1 h2 h3，以后的基本不用
SEO  搜索引擎优化

段落标签：用于表示内容中的一个自然段，使用<p>表示
标签中的文字默认独占一行，段与段之间会有间距
<p>我用来表示一个段落</p>

无论写再多的空格，换行和空格只会当成一个空格解析，使用<br>标签表示换行，<br>是自结束标签
<hr /> 在页面中是生成一条水平线。
<p>

=========  chapter11  图片标签
<img> 向网页中引入外部的图片，也是一个自结束标签
src:设置外部图片的路径
alt:设置在图片不能显示时对图片的描述，搜索引擎可以通过alt属性识别不同的图片
    如果不写alt属性，则搜索引擎不会对img中的图片进行收录
width: 宽度，一般使用 px 作为单位
height: 高度
当宽度和高度只设置其中一个属性时，另一个也会等比例调整大小。
<img src="path" alt="description" width="200px" height="100px"/>

=========  chapter12  相对路径
src属性配置图片的路径，
相对路径：相对于当前资源所在目录的位置。
../ ：返回一级路径，返回几级就写几个

=========  chapter13  图片的格式
JPEG(JPG) ：支持的颜色比较多，图片可以压缩，但不支持透明，一般来保存照片等颜色丰富的图片
GIF ：支持的颜色少，只支持简单的透明（直线的透明），支持动态图，一般图片颜色单一或者动态图时使用gif
PNG ：支持的颜色多，支持复杂的透明，可以用来显示颜色复杂的透明图片，实际开发中用的比较多
图片的使用原则：
    效果不一致：使用效果好的
    效果一致：使用小的

=========  chapter14  meta标签
<meta name="keywords" content="HTML5,JavaScript,Java"/>
name 是对content的描述
（1）可以用来设置网页的关键字，会在搜索结果中展示。
（2）用来指定网页的描述，搜索引擎在检索页面时，会同时检索页面中的关键词和描述，但是这两个值不会影响页面在搜索引擎中的排名。
<meta name="description" content="发布h4,js等前端相关的信息"/>
（3）实现请求的重定向
<meta http-equiv="refresh" content="5;url=http://www.baidu.com"/>
5秒后跳转到百度

=========  chapter15  XHTML语法规范
XHTML语法更加严格一些，仅仅是规范，但不是必须遵守
（1）HTML中不区分大小写，一般都使用小写。
（2）HTML中的注释不能嵌套
    <!--
      不能嵌套注释
      <!--
      -->
    -->
（3）HTML标签必须结构完整，要么成对出现，要么自结束标签
如果缺失标签，浏览器会尽最大努力解析页面。
firebug浏览器插件，查看内存中的页面源码，但是适应于火狐浏览器。
Google  F12开发者工具
浏览器会自动修正html代码。
（4）<br />  <br> 两种书写方式都可以
（5）HTML标签可以嵌套，但不可以交叉嵌套。
（6）HTML标签中的属性必须有值，且值必须加引号，如果不加，浏览器会自动加，单引号和双引号都行

=========  chapter17 18  超链接
可以从一个页面跳转到另一个页面
<a href="demo03.html" target="_self">我是一个超链接</a>
href：指向链接跳转的目标地址，相对路径 或 一个完整的地址
target：指定打开链接的位置
        _self:当前窗口中打开，默认值
        _blank:在新的窗口中打开链接
        也可以在内敛框架中打开，设置为一个内敛框架的name属性值，用的很少。
未访问过：蓝色，访问过的：紫色。
<center> 标签中的内容，默认在页面中居中显示。可以将要居中的元素放在其中。不推荐使用。
<a href="#" /> 如果链接不确定，通过#设置占位符，不会进行跳转
通过截图工具可以测量图片的尺寸大小。
网页 回到顶部 功能： href="#"即可，跳转到当前页面的顶部
去底部 功能：
html中有一个属性，每一个标签都可以设置，id=""，不能以数字开头，id属性在同一个页面中不能重复，仅能有一个。
<a href="#id" /> 直接跳转到id属性所在的位置。
联系我们 功能：直接打开计算机的默认电子邮件的客户端，即 发送电子邮件的超链接，并且设置默认的收件人。

=========  chapter37  列表
1 无序列表
    ul标签，使用li在ul中创建一个一个的列表项，圆点作为项目符号。通过type属性修改无序列表的项目符号。
    ul 和 li 都是块元素
    <ul>
      <li></li>
    </ul>
    去掉项目符号：
    ul{
      list-style:none;
    }

2 有序列表
    使用ol代替ul 其余的与无序列表一样，有序的序号作为项目符号。type属性指定序号的类型

3 定义列表
    对一些词汇或者内容进行定义
    <dl>
      <dt>武松</dt> 被定义的内容
      <dd>景阳冈大虎英雄</dd> 对定义内容的描述
    </dl>
